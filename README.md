# Student Burnout Risk Analyzer

A machine learning project that predicts student burnout risk using behavioral and academic data.

---

## Overview

This project analyzes student-related factors such as study habits, lifestyle, and academic performance to identify patterns associated with burnout. A classification model is built to predict whether a student is at risk.

---

## Features

- Data preprocessing and cleaning  
- Exploratory Data Analysis (EDA)  
- Machine learning model training  
- Burnout risk prediction (Low / High)  

---

## Dataset

The dataset includes features such as:

- Study time  
- Academic performance  
- Social media usage  
- Lifestyle indicators  

---

## Tech Stack

- Python  
- Pandas, NumPy  
- Matplotlib, Seaborn  
- Scikit-learn  
- Jupyter Notebook  

---

## Models Used

- Logistic Regression  
- K-Nearest Neighbors (KNN)  
- Support Vector Machine (SVM)  
- Random Forest
